# Kalouk WEB

::: info ¿Qué es?

Un **servidor web versátil** para generación de datos sintéticos utilizados en el curso y sus proyectos, accesibles por medio de diferentes modelos de comunicación (REST API, WebSocket, SSE y webhooks).

:::
